# Flag artwork

The flag SVGs in this folder come from **flagpedia.net** (served via
**flagcdn.com**). They are drawn in each country's **official aspect ratio**
(e.g. the US at 10:19, Switzerland square, Nepal as its true pennant), so the
flags read as authentic rather than squeezed into a uniform box.

Flagpedia states its flags are free to use for any purpose, including
commercial, without attribution. This note is a courtesy, not a requirement.

- Source: https://flagpedia.net/
- CDN: https://flagcdn.com/
